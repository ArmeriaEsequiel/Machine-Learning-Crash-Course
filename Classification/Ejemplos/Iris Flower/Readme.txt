Este dataset es sobre las caracteristas de las flores de Iris.

las columnas son:
1. sepal length in cm
2. sepal width in cm
3. petal length in cm
4. petal width in cm
5. class:
-- Iris Setosa
-- Iris Versicolour
-- Iris Virginica

Se realizo una clasificacion y se evaluo el modelo. Tambien se hizo uso de KerasClassifier, para poder utilizar metricas como cross_val_score y roc_auc_score.
